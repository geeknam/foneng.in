Push SMS

Making your SMS messaging on Android a breeze.

Send SMS now

Android

Install Push SMS Android application from Android Market. Register the device with downloaded application. Once registered, you will get an invitation on GTalk from pushcontacts@appspot.com. Accept the invitation and you are ready to roll.

Install

GTalk

Receive your incoming SMS through GTalk. You can send replies via GTalk as well. With Push SMS, sending SMS feels just like chatting. Forget about using the touch screen to compose your SMS.

Chrome Extension

You can also install a Chrome Extension from Chrome Web Store. Send your SMS in a blink of an eye with your Chrome browser. This also allows you to receive HTML5 notifications.
